Nintendo Switch 网络问题解决

发表于2022-09-15|更新于2022-10-24|教程

前言

斯普拉顿3发售一个月了，无数的人都在喷任天堂的服务器不行，然而我只是匹配时间稍长（因为会出现连接错误），但基本没出现过游戏中掉线的情况。

网络问题的原因

  • 本节素材来源自百度贴吧 - splatoon吧，@shuilixiang 发表的帖子：【鱿鱼网络研究报告书】SPLATOON3网络研究报告

基本可以归结为：

  1. 2022年9月30日晚到10月5日的联通 CN2 国际出口骨干网出现问题
  2. 部分玩家没有达到 NAT B 的 P2P 游戏基础要求
  3. Splatoon2 的联机对延迟和波动的要求不高，而 Splatoon3 使用的 NPNL 联机 API 提升了准入标准，导致波动较大的网络会出现突然被踢出战局的情况，但只要能正常进入战局，出现通信不安定、严重的射击伤害回馈延迟的情况均有所减轻，总结就是联机体验理论上较前代有所提升
  4. Splatoon3 似乎使用了 NAT 中转，用以提升NAT B~D 玩家的体验，但其本质是一种服务器中转，需要的特定服务商是国际CDN，所以国内环境很难因此受益，反而会因此出现网络问题
  5. 联通网络的玩家联机体验更差基本可以实锤了

网络改善方法

方法一：修改DNS

从PS4早期我就知道的老办法了，可以做到裸连高速下载，缺点是对联机加速没帮助。

可以从路由器上设置或者Switch上设置。

方法二：改善局域网拓扑，使裸连NAT类型为A

登录光猫管理界面 192.168.1.1，使用超级管理员登录，用户名：telecomadmin，密码：nE7jA%5m

选择 应用-高级NAT配置-DMZ设置-输入路由在光猫中的ip（默认为192.168.1.2），保存

登录路由管理界面。在DHCP设置中为Switch固定一个ip地址，然后设置DMZ或端口转发1:65535

方法三：加速器

节省一个加速盒的钱，同时不消耗手机流量和电量。

目前加速盒的原理也就是一个有一定性能的智能路由，需要花费100~200元购买，多占一个排插，还多发射一份WiFi信号。如果有智能路由，如小米、华硕、网件或刷有OpenWRT、梅林等固件的路由，其完全可以担当此任务。

但是很多加速器公司为了推广自家加速盒，没有路由加速这一选项。经过我的测试：

UU主机加速器：支持路由加速，加速效果最好，但OpenWRT系统安装插件比较繁琐（我在之前提到的高通410随身WiFi上没有成功），价格最贵30r/月

灵缇加速器：支持路由加速，加速效果稍好，OpenWRT系统也可一键安装插件，加速下载的速度一般，价格18r/月

方法四：手机热点加速

iOS慎用手机热点加速。

目前对比了几种加速器，感觉除了手机流量和电量消耗快以外，最大的问题是设置繁琐（如果你频繁换加速器）

基本可以分为：设置代理服务；设置主机ip；设置主机ip和代理服务这三大类，繁琐度也是从低到高。对于Switch这种有触屏的掌机来说还好，对于Xbox或者PS5这种只能手柄输入的简直是噩梦。

方法五：主机加速盒

土豪建议不折腾直接买加速盒，省很多事但会把你锁死在一个品牌的加速器上。除非你可以接网线，否则一定要选择有5G频段的主机加速盒。

优点就是多了网线接口，路由上网口少但家里设备多的福音；有WiFi功能，对于打机位置离家中主路由远，信号差的，可以把加速盒离打机位置仅一些。

雷神比较便宜，如果没有智能路由，那他家的加速盒性价比还挺高。
